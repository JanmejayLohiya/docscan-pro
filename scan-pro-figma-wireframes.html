ScanPro — wireframe frames

Ten screens as standalone mobile artboards (360 × 780), stripped of device bezels and annotations so they import into Figma as clean, editable frames. Grayscale wireframe fidelity; the single accent marks primary actions and active states.

Import to Figma: install the free html.to.design plugin → Import via URL → paste this page’s published URL. Each 360 × 780 block imports as its own frame. Then set frame names, turn repeated parts (tab bar, buttons) into components, and layer on visual design. A high-fidelity version already exists in figma-design.html.
