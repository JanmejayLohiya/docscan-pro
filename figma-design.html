Visual design — Figma-ready

Production-fidelity screens with a real palette, type scale, icon set, and components. Import straight into Figma as editable layers using the steps below, then wire up interactions and components there.

Import into Figma (html.to.design)

  1. In Figma, open Plugins → Browse plugins and install “html.to.design” (free).
  2. Run the plugin, choose Import via URL, and paste this page’s published artifact URL.
  3. The plugin imports every screen as editable frames, text, and vector icons — grouped and named.
  4. In Figma, select repeated elements (tab bar, buttons, cards) and Create component to build your library.
  5. Tokens below map 1:1 to Figma variables/styles — recreate them once and rebind. Also in figma-design.html.
